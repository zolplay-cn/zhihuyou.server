<h1 align="center">知忽悠 Zhihuyou Server</h1>

- [Nest](https://github.com/nestjs/nest) 7
- TypeScript 4.

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn dev

# migrate db
$ yarn migrate:dev

# production mode
$ yarn start:prod
```

## Test

```bash
# unit tests
$ yarn test

# e2e tests
$ yarn test:e2e
```

## Duplicate .env
```bash
cp .env.example .env
cp .env.example .test.env
```
